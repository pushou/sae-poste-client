# sae-poste-client

Votre responsable vous demande d'étudier avec un autre technicien un panel de solutions d'installation automatisée des postes clients windows et de Linux.
Il vous est demandé d'étudier les solutions suivantes:
- serveur PXE pour installation de serveurs Linux
- couple "WDS + MDT" reste opérationnel avec la dernière version en date de Windows 11
- solution pré-packagée FOG
- iventoy
- wapt 
A la fin il vous est demandé de rédiger un tableau synthétiqueet comparatif des solutions
Afin de travailler dans de bonnes conditions vous vous isolerez sur un LAN dédiée.
